# Backend-Experiment1: Simple Database Connection & Display

## Steps

### 1. Import Database
Import `users_table.sql` in phpMyAdmin.

### 2. Place Files
Put all files in:
MAMP: /Applications/MAMP/htdocs/experiment1

Files:
db_connect.php
display_users.php
users_table.sql

### 3. Run
MAMP:
http://localhost:8888/experiment1/display_users.php
